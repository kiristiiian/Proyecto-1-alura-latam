# Análisis de Datos – Desafío Alura Store

## Descripción del proyecto

Este proyecto forma parte de un desafío de **análisis de datos** cuyo objetivo es ayudar al Sr. Juan a tomar una decisión estratégica: **determinar qué tienda de la cadena Alura Store debería vender** para iniciar un nuevo emprendimiento.

Para ello, se analizan datos de **ventas, rendimiento y reseñas de clientes** correspondientes a **cuatro tiendas** de Alura Store, utilizando herramientas de análisis y visualización de datos en Python.

El enfoque del proyecto está pensado para demostrar habilidades prácticas en **manipulación de datos, generación de gráficos e interpretación de insights**, orientado a un contexto real de negocio.

---

## Objetivo

Identificar la **tienda menos eficiente** a partir de métricas clave y presentar una **recomendación final basada en datos**, respaldada por visualizaciones claras y conclusiones argumentadas.

---

## Tecnologías y librerías utilizadas

* **Python**
* **Pandas** – carga y manipulación de datos CSV
* **Matplotlib** – visualización de datos
* **Jupyter Notebook / Google Colab**

---

## Análisis realizado

Durante el desarrollo del notebook se trabajan los siguientes aspectos:

* Ingresos totales por tienda
* Categorías de productos más vendidas
* Productos con mayor volumen de ventas
* Reseñas y calificaciones de clientes
* Costos promedio de envío

A partir de este análisis se obtienen **insights clave** que permiten comparar el rendimiento de cada tienda.

---

## Visualizaciones

El proyecto incluye un **mínimo de tres gráficos diferentes**, entre ellos:

* Gráficos de barras para comparar ingresos y ventas
* Gráficos circulares para distribución por categorías
* Gráficos de dispersión u otros para evaluar rendimiento y reseñas

Estas visualizaciones facilitan la interpretación de los datos y refuerzan la recomendación final.

---

## Recomendación final

Con base en los datos analizados y las visualizaciones generadas, se presenta una **recomendación argumentada** sobre **qué tienda debería vender el Sr. Juan**, explicando claramente los motivos desde una perspectiva de negocio y eficiencia.
